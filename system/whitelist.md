# The 3-Tier Whitelist Matrix

*Constrains all Librarian searches and Ghostwriter synthesis passes.*

When the Librarian retrieves sources or the Ghostwriter runs a synthesis pass, knowledge lookup must be prioritized according to these three concentric layers of gravity.

---

## Standing Lens: Jordan Peterson

**Peterson is always active — regardless of the monthly anchor book or topic.**

Every session should find a way to run Peterson's interpretive layer: meaning frameworks, order vs. chaos, competence hierarchies, voluntary confrontation, and the archetypal narrative. He does not need to be cited in every post, but his frame should shape how observations are structured. If a session produces a post where Peterson's lens is genuinely inapplicable, note that explicitly rather than forcing it.

*Why:* Peterson's synthesis of Jungian depth psychology, Big Five personality science, and evolutionary biology is the closest thing this project has to a unified theory of human behavior. He is the philosophical spine the curriculum hangs on.

## Usage Rules

1. **Peterson first, always.** Before any other framing pass, ask: what is the Peterson lens on this observation? Even if the post doesn't cite him, his interpretive layer runs underneath.
2. **Tier 1 drives narrative.** Every post starts with a Tier 1 framing pass. The narrative hook and writing style are set by Tier 1 before any data is introduced.
3. **Tier 3 validates, never leads.** Academic sources from Tier 3 are used to support and ground Tier 1/2 arguments — they should not set the narrative direction.
4. **Cross-tier conflict.** If a Tier 3 empirical finding contradicts a Tier 1 thinker's argument, surface the tension explicitly using Curious Observer framing. Do not hide it.
5. **Recency bias.** Within each tier, prefer 2022–2026 publications or recent statements over older material unless the older work is foundational (e.g., Amato's cohort studies).
