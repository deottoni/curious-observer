# The Curious Observer — Project Context

A 6-month pre-master's psychology research and writing project. Digital lab notebook and personal portfolio. Not a brand, not a side hustle.

## What This Is
A structured intellectual experiment to test whether Andre genuinely enjoys academic research methodology — or just the high-level ideas. Twelve posts over six months is the trial. The outcome determines whether a formal Psychology Master's is the right next move.

## Session Protocol
1. Read the most recent file in `sessions/` to orient on current state
2. Read `system/blueprint.md` for tone rules and agent pipeline
3. Ask which post or phase we're working on
3. Follow the agent pipeline in order: Interrogator → Librarian → Auditor → Ghostwriter → *(Audio Producer: planned, not active)*
4. Audio Producer is planned but NOT active — do not initialize it; see `system/agents/audio-producer.md` for activation checklist

## Hard Constraints
- Never skip the Interrogator step — no synthesis before the hypothesis is sharpened
- All posts must cite 2022–2026 sources where possible
- Every post must comply with Curious Observer tone rules (see `system/blueprint.md`)
- Andre learns by: concept first → worked example second. Never throw a raw paper at him cold.
